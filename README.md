# DwFramework
## 0x1 项目简介
基于Autofac的NetCore快速开发框架，这个框架旨在将服务注入简单化，把Autofac中常用的部分暴露出来，并融合了其他几个项目开发常用的组件。让整个开发的过程变得简单快速。当然，如果你有更复杂的业务需求，你可以直接引用Autofac来对本框架进行扩展。
